This Github is a repository of scripts, presets, and project files used for the DS9: Redefined  and VOY: Redefined restoration and upscale projects.

For seasons 1-3 of DS9 and seasons 1-4 of VOY, these file utilize the LaserDisc stacks (not copies) here: https://archive.org/details/@section_31

The rest of the seasons use the American NTSC DVDs.
